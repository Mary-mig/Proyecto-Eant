Aprendiendo Git

Git es un software de control de versiones diseñado por Linus Torvalds.

 

Ayer tuvimos una nueva reunión del grupo de usuarios git (la última de este 2014 que se acaba).

En esta ocasión Oscar Vitores compartió con nosotros su experiencia en el uso de git-flow. La presentación fue muy interesante ya que no sólo se limitó a exponer qué es y cómo funciona, sino que nos contó cómo la usa con su equipo, qué le sirvió y cuándo git-flow empezó a quedarse corto.

El meetup estuvo muy bien, con muchas preguntas y comentarios durante la exposición. Gracias a todos los que estuvisteis allí y a Oscar por venir a contarnoslo en persona.
